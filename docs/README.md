## 目录规范

每个 `phaseN/` 文件夹应包含：

```
phaseN/
├── README.md       # 阶段索引与状态
├── 开发文档.md      # 面向 AI / 开发者的详细规格（Props、路由、验收标准等）
└── 实施计划.md      # 分步骤实施计划与进度跟踪
```

新增阶段时，复制 `phase1/README.md` 模板并补充对应文档即可。
