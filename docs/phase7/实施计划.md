## 后端依赖

| 模块 | 主要路径 |
|------|---------|
| Dashboard | `GET /api/dashboard/stats`、`GET /api/dashboard/token-trend` |
| API 发布 | `POST /api/workflows/:id/publish`、`GET /api/published-apis`、`POST .../reset-key`、`PATCH`、`DELETE` |
| 全局搜索 | `GET /api/search?q=&limit=` |
| 近期列表回退 | `GET /api/workflows`、`GET /api/executions`（统计端点不可用时部分聚合） |
