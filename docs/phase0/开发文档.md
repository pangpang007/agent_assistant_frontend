> 本文档面向 Cursor / AI 编程助手，目标是根据本文档直接生成可运行的前端代码。所有样式描述均引用 design token，禁止硬编码色值。

## 1. 目标

搭建「汤圆代码助手」Phase 0 前端工程：建立 GitHub Dark 风格设计系统、项目脚手架、基础组件库（13 个组件）、路由框架与布局骨架、主题切换能力，为后续 Phase 1 的工作流编辑器等核心功能提供完整基础设施。

### 4.3 主题切换机制

**原理**：通过 `<html data-theme="dark|light">` 属性切换。

**Zustand store（themeStore.ts）**：

```ts
// stores/themeStore.ts 的逻辑说明
interface ThemeState {
  theme: 'dark' | 'light';
  setTheme: (theme: 'dark' | 'light') => void;
  toggleTheme: () => void;
}
```

- 初始化时读取 `localStorage.getItem('theme')`，若无则使用 `window.matchMedia('(prefers-color-scheme: dark)').matches` 推断系统偏好
- `setTheme` 方法同时：① 设置 `document.documentElement.setAttribute('data-theme', theme)`；② 写入 `localStorage.setItem('theme', theme)`
- `ThemeProvider` 组件在 mount 时执行 DOM 属性初始化，避免闪烁（在 `<head>` 中嵌入一段内联 script 同步设置 data-theme）

**防闪烁脚本（index.html `<head>` 中）**：

```html
<script>
  (function() {
    const theme = localStorage.getItem('theme') ||
      (window.matchMedia('(prefers-color-scheme: dark)').matches ? 'dark' : 'light');
    document.documentElement.setAttribute('data-theme', theme);
  })();
</script>
```

### 5.1 Button

**文件路径**：`src/components/ui/Button/`

#### Props 定义

```ts
type ButtonVariant = 'primary' | 'secondary' | 'danger' | 'ghost';
type ButtonSize = 'sm' | 'md' | 'lg';

interface ButtonProps extends React.ButtonHTMLAttributes<HTMLButtonElement> {
  variant?: ButtonVariant;       // 默认 'secondary'
  size?: ButtonSize;             // 默认 'md'
  loading?: boolean;             // 加载中状态
  leftIcon?: React.ReactNode;    // 左侧图标
  rightIcon?: React.ReactNode;   // 右侧图标
  fullWidth?: boolean;           // 是否撑满容器宽度
  children: React.ReactNode;
}
```

#### 样式规范

| 属性 | primary | secondary | danger | ghost |
|------|---------|-----------|--------|-------|
| 背景色 | `var(--accent-primary)` | `transparent` | `var(--accent-danger)` | `transparent` |
| 文字色 | `var(--text-inverse)` | `var(--text-primary)` | `#ffffff` | `var(--text-primary)` |
| 边框 | 无 | `1px solid var(--border-default)` | 无 | 无 |

| 尺寸 | 高度 | 水平 padding | 字号 | 圆角 |
|------|------|-------------|------|------|
| sm | 28px | var(--space-3) | var(--text-xs) | var(--radius-sm) |
| md | 32px | var(--space-4) | var(--text-sm) | var(--radius-md) |
| lg | 40px | var(--space-5) | var(--text-base) | var(--radius-md) |

#### 交互行为

| 状态 | 表现 |
|------|------|
| **hover** | primary: 背景色变为 `var(--accent-primary-hover)`；secondary: 背景变为 `var(--bg-hover)`，边框变为 `var(--border-strong)`；danger: 背景变为稍亮红；ghost: 背景变为 `var(--bg-hover)` |
| **active** | 背景色变为各对应的 `-active` 色或 `var(--bg-active)` |
| **focus** | 显示 `var(--shadow-focus)` 外发光环 |
| **disabled** | `opacity: 0.5; cursor: not-allowed; pointer-events: none;` |
| **loading** | 文字变透明（仍占位），左侧显示小尺寸 Spinner，`pointer-events: none` |

#### 动效

- 所有颜色变化：`transition: all var(--duration-normal) var(--ease-default)`
- loading spinner 使用 `rotate` 无限动画
